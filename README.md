## ⚡ Quick Start

```bash
npm install @randaotoken/component-library
```

```tsx
import { Button, ThemeProvider } from '@randaotoken/component-library';

function App() {
  return (
    <ThemeProvider>
      <Button variant="primary">
        Click me
      </Button>
    </ThemeProvider>
  );
}
```
